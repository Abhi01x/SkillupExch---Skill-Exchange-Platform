## 📌 Overview

**SkillExchange** is a modern peer-to-peer learning platform where students can **exchange skills instead of money**.
Users can connect, teach what they know, and learn what they don’t — creating a collaborative ecosystem of growth.

This project focuses on **real-world impact**, combining **full-stack engineering + scalable architecture + modern UI/UX**.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repo

```bash
git clone https://github.com/Abhi01x/SkillupExch---Skill-Exchange-Platform.git
cd SkillupExch---Skill-Exchange-Platform
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
```

Create `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
```

Run backend:

```bash
npm run dev
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```
